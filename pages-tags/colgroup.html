<Colgroup>

Exemplo

Descrição

<colgroup>

<col span="2" style="background-color:red">

<col style="background-color:yellow">

</colgroup>

a tag é útil para aplicar estilos a colunas inteiras, em vez de repetir os estilos para cada célula, para cada linha.

Nota: a tag deve ser filha de um elemento <table>, depois de qualquer elemento <caption> e antes de qualquer elemento <thead>, <tbody>, <tfoot> e <tr>.

Dica: para definir propriedades diferentes para uma coluna dentro de um <colgroup>, use a tag <col> dentro da tag <colgroup>.
