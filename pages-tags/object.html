<object>

Exemplo

Descrição

<object data="video.mp4" width="400" height="300"></object>

é uma tag HTML usada para exibir multimídia como áudios, vídeos, imagens, PDFs e Flash em páginas da web.

também pode ser usado para exibir outra página da web dentro da página HTML.

qualquer texto escrito entre as tags e é considerado um texto alternativo que aparece quando os dados especificados não são suportados pelo navegador.
