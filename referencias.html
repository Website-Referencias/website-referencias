Abaixo temos algumas tags e sua descrição:

Tag

Descrição

<colgroup>

especifica um grupo de uma ou mais colunas em uma tabela para formatação

<pre>

define texto pré-formatado

<i>

define uma parte do texto com uma voz ou humor alternativo. O conteúdo interno normalmente é exibido em itálico

<source>

é usada para especificar vários recursos de mídia para elementos de mídia

<li>

define um item da lista

<footer>

define um rodapé para um documento ou seção

<input>

especifica um campo de entrada onde o usuário pode inserir dados

<s>

especifica o texto que não é mais correto, preciso ou relevante. O texto será exibido com uma linha atravessando-o

<span>

é um contêiner embutido usado para marcar uma parte de um texto ou parte de um documento

<object>

define um contêiner para um recurso externo

<p>

define um parágrafo

<video>

é usada para incorporar conteúdo de vídeo em um documento, como um clipe de filme ou outros fluxos de vídeo
